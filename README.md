## About

Synka is India's most powerful digital business card platform with NFC & QR sharing, built-in CRM, and AI-designed card templates.

## Features

- 1-tap digital card exchange (NFC + QR)
- Automatic lead capture
- Built-in CRM for contacts & notes
- AI-designed cards & templates
- Real-time analytics
- Team & employee card tracking

## Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase (Backend)
- Capacitor (Mobile)

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Mobile Apps

This project uses Capacitor for iOS and Android builds.

```sh
# Sync native projects
npx cap sync

# Open in Android Studio
npx cap open android

# Open in Xcode
npx cap open ios
```
